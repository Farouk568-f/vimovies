ViMovies Changelog

Release notes for ViMovies. Every version is published here and on the download page at the same time. Install updates over the existing app — your data carries over.

Version 2.4 — July 2026 (current)

  • New download manager with pause/resume and USB storage targets on TV devices.
  • Faster source resolution — most titles now start 30–40% quicker.
  • Subtitle size and background options in the player.
  • Fixed: focus loss on the genre rail after returning from playback.

Version 2.3 — May 2026

  • Anime section reorganized with season grouping and airing status.
  • Watchlist reordering with the remote's long-press.
  • Reduced app size by ~8 MB via image pipeline optimization.
  • Fixed: resume position occasionally off by a few seconds on long films.

Version 2.2 — March 2026

  • Initial Google TV interface refinements (rounded focus states, denser rails).
  • Adaptive playback quality on unstable connections.
  • Fixed: crash on Android 7 devices when opening search.

Version 2.1 — January 2026

  • First public Android TV build with the remote-first interface.
  • Offline downloads for movies and episodes.
  • On-device watch progress and watchlist.
Running into problems after an update? Clear the app cache first (Settings → Apps → ViMovies → Clear cache), then check the troubleshooting guide.

Frequently Asked Questions

How do I update to the latest version?

Download the newest APK from the download page and install it over your current version. Your watchlist, settings, and progress are preserved.

How often is ViMovies updated?

Updates ship every few weeks, alternating between feature releases and smaller fix rounds.

Do I have to update?

Older versions keep working, but updating is recommended — source reliability and player fixes land in almost every release.
