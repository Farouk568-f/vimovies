Install on Android TV or Google TV (Short Version)

  1. Open a browser on your TV (or use Send Files to TV from your phone) and download the APK from the download page.
  2. Open the file from Downloads. When prompted, allow your browser or file manager to install unknown apps.
  3. Select Install. ViMovies appears in your apps row when it finishes.

Install on Fire TV Stick (Short Version)

  1. Install Downloader from the Amazon Appstore.
  2. Enable Apps from Unknown Sources for Downloader in Settings → My Fire TV → Developer Options.
  3. Enter the ViMovies download URL in Downloader and install the APK when it finishes.

Install on an Android Phone or Tablet

  1. Open this site in Chrome and tap the download button.
  2. Tap the finished download notification.
  3. Allow Chrome to install unknown apps if asked, then tap Install.

Frequently Asked Questions

How long does installation take?

About five minutes on any device. The APK is small, and the only extra step is granting the one-time permission to install apps from outside the Play Store.

Do I need a computer?

No. Everything happens on the TV or phone itself using a browser or the Downloader app.

Is sideloading safe?

Sideloading itself is just installing an app manually. It is safe as long as the APK comes from the developer's official site — which is exactly what this page links to.

The install button is greyed out. What now?

This usually means the unknown-sources permission was not granted to the app you opened the APK with. Grant it in Settings, then reopen the file. More fixes are in our troubleshooting guide.
